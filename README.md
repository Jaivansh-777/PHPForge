# PHPForge 🚀

**PHPForge** is a modern browser-based PHP Compiler & Runner built using PHP, HTML, CSS, JavaScript, Docker, and cloud deployment tools.  
Created by **Jaivansh**, it allows users to write, test, validate, and run PHP code directly from the browser.

---

## 🌐 Live Demo

🔗 https://phpforge-1.onrender.com/

---

## ✨ Features

- ⚡ Run PHP code instantly
- ✅ Syntax checking support
- 🎨 Clean and professional UI
- 📱 Responsive design (Mobile + Desktop)
- 🧠 Built-in PHP code snippets
- ⏱ Execution time tracking
- 🛡 Secure temporary file handling
- 🌙 Dark code editor theme
- ☁️ Cloud deployed with Docker + Render
